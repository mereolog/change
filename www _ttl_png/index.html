Nieszawa case study

To illustrate the issue more palatable we will use in this paper the following running example. The contemporary city of Nieszawa was first mentioned in historical records around 1425 as located at N52o59.48' E18o30.28' - we will call this location location 1. Around 1460 Nieszawa was relocated to N52o50.12' E18o54.05' - we will call this location "location 2". In short:

  1. Nieszawa is located at location 1 from 1425 to 1459.
  2. Nieszawa is located at location 2 from 1460 to the present day.

In each solution Nieszawa is represented as the OWL individual ":settlement1" and the two locations as ":location1" and ":location2". Using this minimal data set each pattern was implemented as an OWL ontology for the running example.
